### 🎨 文案优化
- 时间锁点击提示改为**逐字出现**（80ms/字）+ 闪烁光标动画
- 时间锁文字：**「就算你是我的公主大人，也不能直接偷看哦 💕 还有 X天X小时…」**

- 完整文案库收录到 README 上方「调情文案库」

### 🚀 性能优化
- 背景**流星改为 Canvas 渲染**（之前是 DOM + CSS box-shadow + gradient，每次出现触发重绘）
- **FairyDustCursor** CDN 库替换为自研轻量版（限速 60ms，自动消除）
- **鼠标光晕**改用 `transform` + `will-change` 代替 `left/top`（避免重排 → GPU 合成）
- **背景粒子**生成频率从 400ms 降至 600ms
- Gate 页**极光 blur：80px→50px** | 星云 blur：60px→40px
- **樱花 canvas 降频**：60fps → 约 30fps，延迟移除即时启动
- 移除不再使用的 CDN：`canvas-confetti` + `cursor-effects`
- 用 `sparkleBurst()` 替代 `confetti()`（原库在微信浏览器可能白屏）

### 🎯 其他调整
- 时间锁晃动**幅度 6px→3px，频率降低，时长 0.5s→0.6s**（更轻柔）
- 点击流星特效**扇形散开 45° + 飞行距离加倍 + 速度降低**
- 背景流星尾焰方向修复（之前画反了）
